# User Profile Card Generator

A Flask-based web application that allows users to create professional profile cards by entering their personal information.

## Features

* User Name Input
* Email Input
* Bio Section
* Skills Section
* LinkedIn Profile Link
* GitHub Profile Link
* Profile Image URL
* Dynamic Profile Card Generation
* Download Profile Card Feature
* Responsive Design
* Dark Mode UI

## Technologies Used

* Python
* Flask
* HTML5
* CSS3
* JavaScript

## Installation

```bash
pip install -r requirements.txt
python app.py
```

## Run

Open:

http://127.0.0.1:5000

## Project Objective

This project demonstrates frontend-backend communication using POST requests, form handling, dynamic rendering, and responsive web design.
